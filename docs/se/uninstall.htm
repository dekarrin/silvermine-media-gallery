Coppermine Photo Gallery v1.5.x: Dokumentation och Manual

Innehållsförteckning

Avinstallering

Om du inte längre vill ha ditt Coppermine galleri och vill göra dig av med det, behöver du göra två saker:

  • Radera databastabellerna

    Använd ett tredjepartsverktyg som phpMyAdmin för att radera alla coppermine tabeller (se skärmbild). Du kan avgöra vilka tabeller du behöver ta bort genom att titta på tabellens prefix: alla coppermine-tabeller använder det tabellprefix som du valde när du körde installation för första gången. Om du inte är säker på vad du gör, gör en backup (database dump) före du slutgiltigt raderar tabellerna. Som standard är tabellprefixet cpg15x_ (men kan vara annorlunda om du valde ett annat prefix under installationen). När du använder phpMyAdmin, bocka för alla coppermine-tabeller, bläddra sedan till slutet av sidan och välj "drop" från rullgardinsmenyn märkt "with selected". Bekräfta säkerhetsdialogen med ja, "yes".
    Alternativt (om du inte har phpMyAdmin tillgängligt), skapa en PHP fil (med hjälp av en ren-text editor som notepad.exe) med detta innehåll: 
    <?php
// add the mysql connection stuff here

DROP TABLE `cpg15x_albums`, `cpg15x_banned`, `cpg15x_bridge`, `cpg15x_categories`,
`cpg15x_comments`, `cpg15x_config`, `cpg15x_dict`, `cpg15x_ecards`, `cpg15x_exif`,
`cpg15x_favpics`, `cpg15x_filetypes`, `cpg15x_hit_stats`, `cpg15x_pictures`, `cpg15x_plugins`,
`cpg15x_sessions`, `cpg15x_temp_data`, `cpg15x_temp_messages`, `cpg15x_usergroups`,
`cpg15x_users`, `cpg15x_votes`, `cpg15x_vote_stats`;
?>
    spara filen som "droptables.php" och ladda upp den (med hjälp av din FTP applikation) till den mapp där ditt coppermine galleri finns. Kör sedan filen du just laddade upp i din webbläsare genom att gå till URL http://din_hemsida.se/din_gallerimapp/droptables.php
  • Radera filerna
    Använd din FTP applikation för att radera den mapp dit du installerade coppermine galleriet.
