Innehållsförteckning

Uppladdning med HTTP

Vanliga HTTP uppladdningar använder webbläsarens inbyggda möjligheter att ladda upp filer till servern. Den maximala filstorleken bestäms av två grundläggande faktorer: den hastighet och den datamängd som webbläsaren kan ladda upp med innan tidsgränsen går ut, och den tillåtna filstorleken som bestäms av serverns inställningar. Notera att dessa inställningar inte är bestämda av Coppermine, utan av serverkonfigurationen (php.ini). Användare som har webbhotell kan vanligtvis inte redigera php.ini, så de får leva med dom begränsningar som webbserverns administratör har satt upp. Dom som faktiskt kör sin egen server och kan redigera php.ini bör titta på de inställningar som förklaras på felsökningssidan för uppladdningar om (stora) http uppladdningar misslyckas.

Det torde vara uppenbart att filer måste laddas upp någonstans (till någon mapp) på din webbserver - detta är mappen "albums" inom den mapp som du installerade Coppermine i på din server. HTTP uppladdningar går till en undermapp av "userpics" mappen (som finns i "albums" mappen). Naturligtvis behöver Coppermines uppladdningsprogram behörigheter till mappen för att kunna ladda upp filer dit. Det är därför du måste ändra behörigheter på albums mappen och allting inom den under installationen av Coppermine - göra den skrivbar för den användare som webbservern körs med. Detta görs genom att använda kommandot CHMOD på en Unix/Linux baserad server. Om du får problem under uppladdningen, kontrollera att du satt behörigheterna rätt.

Uppladdningsgränssnitt

Coppermine kommer med två uppladdningsgränssnitt: en traditionell som använder enkla HTML metoder för att ladda upp filer en och en och en avancerad, flash-driven, som gör det möjligt att ladda upp flera filer på en och samma gång.

Flash-drivet gränssnitt för uppladdning av flera filer

Det flash-drivna gränssnittet erbjuder fler möjligheter och är lättare att använda, därför rekommenderar vi att det används som standard.

Användning

  1. Välj ett album från listrutan
  2. Klicka på "Bläddra" knappen och navigera till den fil som du vill ladda upp
  3. Välj fler filer att ladda upp genom att repetera steg 3
  4. Klicka på "Fortsätt" knappen efter att du laddat upp alla filer du valt (Knappen kommer att synas efter att du ladda upp minst en fil).
  5. Du kommer att skickas till en skärmbild där du kan skriva in mer detaljer om dina uppladdade filer. Efter att du fyllt i det, skicka formuläret genom att använda "Spara ändringar" knappen i slutet av formuläret.

HTML-drivet uppladdningsgränssnitt för enstaka filer

Det HTML-drivna uppladdningsformulär för enstaka filer rekommenderas för de som inte har Flash installerad (av någon anledning) och särskilt vid problem med uppladdning.

Användning

  1. Välj ett album från listrutan list
  2. Klicka på "Bläddra" knappen och navigera till den fil som du vill ladda upp
  3. Ange filens titel, beskrivning och nyckelord i därför avsedda fält om du så vill
  4. Klicka på "Ladda upp fil" knappen för att skicka formuläret och faktiskt ladda upp filen.
