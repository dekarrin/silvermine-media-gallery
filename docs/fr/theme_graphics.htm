Ressources graphiques concernant les thèmes

Les ressources graphiques à l'intérieur des thèmes peuvent être personnalisées et configurées. Les standards suivants sont utilisés pour changer ou modifier l'aspect de votre thème personnalisé:

  • Icônes de menu
  • Images de la barre de navigation
  • Images de notation
  • Image du négatif de film
  • Image de progression (chargement)

Icônes de menu

Le terme icônes de menu peut être un peu trompeur: il dépend de l'option de configuration correspondante "activer les icônes de menu" Il n'y a pas que les icônes du menu administrateur qui peuvent être prisent en compte, mais toutes les icônes au travers de toutes les pages de Coppermine, aussi bien pour les administrateurs que pour les autres visiteurs pour "relever" ou pas avec une petite icône représentant l'action qui peut être faite ou qui a été faite.

Un thème pour Coppermine peut contenir son propre jeu d'icônes. S'il ne contient pas son propre jeu d'icônes, c'est le jeu d'icônes par défaut se trouvant dans le répertoire ./votre_repertoire_coppermine/images/icons/ qui est utilisé. Si un thème contient son propre jeu d'icônes, les icônes se trouvent dans le répertoire ./votre_repertoire_coppermine/themes/votre_theme_personnalise_theme/images/icons/. Vous devez explicitement spécifier qu'un thème personnalisé à son propre jeu d'icônes. Pour cela, vous devez ouvrir ./votre_repertoire_coppermine/themes/votre_theme_personnalise_theme/theme.php avec un éditeur de texte et ajouter/dé-commenter une ligne comme ceci:

define('THEME_HAS_MENU_ICONS',16);

Si une telle ligne est présente, Coppermine cherchera les icônes de menu dans ./votre_repertoire_coppermine/themes/votre_theme_personnalise_theme/images/icons/. Si une telle ligne n'est pas présente, le jeu d'icônes par défaut sera utilisé. Le code que vous devez ajouter doit se trouver dans une ligne propre. Elle défini une constante et remplis celle-ci avec la valeur "16", qui sera la valeur pour la taille (en pixels) du côté de l'icône de menu.

La fonction icônes de menu est relativement nouvelle, donc, avec la publication initiale de cpg1.5.x, il n'y aura probablement pas beaucoup de choix dans les icônes de menu. Mais cela peut changer et le nombre de jeux d'icônes augmenter. Il est recommandé de vérifier le site officiel de temps en temps pour de nouveaux jeux d'icônes.

Images de la barre de navigation

Un thème pour peut contenir un jeu d'images utilisé pour la barre de navigation. La barre de navigation graphique est la ligne au dessus des images intermédiaires qui permets aux visiteurs de passer à l'image précédente, à la suivante d'un album et (si disponible) d'afficher ou de cacher la section des informations de l'image, et permets aux utilisateurs de poster des commentaires et de lancer le diaporama. Si le thème n'a pas de fichiers graphiques propres pour la barre graphique de navigation, c'est le jeu d'images par défaut se trouvant dans le répertoire ./votre_repertoire_coppermine/images/navbar/ qui est utilisé. Si un thème contient un jeu d'images pour la barre de navigation, ces images se trouvent dans le dossier ./votre_repertoire_coppermine/themes/votre_theme_personnalise/images/navbar/. Vous devez spécifier explicitement que votre thème personnalisé a son propre jeu d'images. Pour cela, vous devez ouvrir ./votre_repertoire_coppermine/themes/votre_theme_personnalise/theme.php avec un éditeur de texte et ajouter/dé-commenter une ligne comme ceci:

define('THEME_HAS_NAVBAR_GRAPHICS',1);

Si une telle ligne est présente, Coppermine va cherche les images pour la barre de navigation dans ./votre_repertoire_coppermine/themes/votre_theme_personnalise/images/navbar/. Si une telle ligne n'existe pas, les images par défaut de Coppermine seront utilisées. Le code que vous devez ajouter doit être placé dans une ligne propre. Elle défini une constante et la remplie avec la valeur "1", valeur qui n'a pas de signification particulière.

Si vous décidez de créer vos propre images pour la barre de navigation de votre thème personnalisé, les images suivantes doivent se trouver dans le répertoire de votre barre de navigation personnalisée (./votre_repertoire_coppermine/themes/votre_theme_personnalise/images/navbar/):

  • ecard.png
  • info.png
  • next.png
  • prev.png
  • report.png
  • slideshow.png
  • thumbnails.png

Images de notation

Un thème pour Coppermine peut contenir des images pour la notation. La possibilité de permettre aux visiteurs d'évaluer les images n'est pas lié à la présence des images de notation - c'est un paramètre qui dépend des permissions des groupes de visiteurs dans (→ Panneau de contrôle des groupes) et des propriétés des albums (→ Propriétés de l'album). Si un thème n'a pas d'images de notation propre, c'est le jeu d'images par défaut qui se trouve dans le dossier ./votre_repertoire_coppermine/images/. Si un thème a ses propres images de notation, elles se trouvent dans le dossier ./votre_repertoire_coppermine/themes/votre_theme_personnalise/images/. Vous devez explicitement définir qu'un thème personnalisé à ses propres images de notation. Pour cela, vous devez ouvrir ./votre_repertoire_coppermine/themes/votre_theme_personnalise/theme.php avec un éditeur de texte et ajouter/dé-commenter une ligne comme ceci:

define('THEME_HAS_RATING_GRAPHICS',1);

Le code que vous devez ajouter doit être placé dans une ligne propre. Elle définie une constante, lorsque la valeur qui n'a pas de signification particulière.

Si elle est définie, les fichiers suivants doivent se trouver dans le répertoire images du thème:

  • rate_empty.png
  • rate_full.png
  • rate_new.png
  • rating.png

Image du négatif de film

Un thème pour Coppermine peut contenir un jeu d'images utilisées pour composer le négatif de film. Si un thème n'a pas ses propres images de négatif de film, ce sont les images par défaut qui se trouvent dans le dossier ./votre_repertoire_coppermine/images/. qui sont utilisées. Si un thème a son propre jeu d'images pour le négatif de film, les images se trouvent dans le dossier ./votre_repertoire_coppermine/themes/your_custom_theme/images/. Par défaut, l'image utilisée à pour nom tile.gif.

Vous devez explicitement spécifier qu'un thème a son propre jeu d'images pour le négatif de film. Pour cela, vous devez ouvrir ./votre_repertoire_coppermine/themes/votre_theme_personnalise/theme.php avec un éditeur de texte et ajouter/dé-commenter une ligne comme ceci:

define('THEME_HAS_FILM_STRIP_GRAPHIC',1);

Le code que vous devez ajouter doit être placé dans une ligne propre. Elle définie une constante, lorsque la valeur qui n'a pas de signification particulière.

Image de progression (chargement)

Un thème pour Coppermine peut contenir une image utilisée comme barre de progression ou de l'icône de chargement. Avec des applications web modernes les utilisateurs attendent de voir de la page qu’ils utilisent si elle bien occupés à quelque chose (soit un chargement ou un téléchargement) - c'est ce quoi la barre de progression est utilisé.
Si un thème n’a pas sa propre image de progression, l'image par défaut est utilisée, elle réside dans le dossier ./votre_repertoire_coppermine/images/. Si un thème est livré avec une image de progression, l'image va résider dans le dossier ./votre_repertoire_coppermine/themes/votre_theme_personnalise/images/. Par défaut, l'image de progression est nommée loader.gif.

Vous devez explicitement préciser qu’un thème a sa propre image de progression. Pour ce faire, vous devez ouvrir ./votre_repertoire_coppermine/themes/votre_theme_personnalise/theme.php avec un éditeur de texte et ajouter/dé-commenter une ligne comme ceci:

define('THEME_HAS_PROGRESS_GRAPHICS',1);

Le code que vous devez ajouter doit être placé dans une ligne propre. Elle définie une constante, lorsque la valeur qui n'a pas de signification particulière.
